Instructions

Use the JSON.parse() method to read the key ex-06-data from local storage and then create an li element for each element in the data array.

The li elements should be added as child elements of the .ex-list ul element.

The li element should have the text content of the name property in each object.

The ex-06-data array is an array of objects:

[{ name: "Barcelona" }, { name: "Madrid" }, { name: "Sevilla" }]

The JSON.parse(string) method can be used to convert JSON to an object.
